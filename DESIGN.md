# Sistema de Diseño de Legado

## Visión General
Legado es una plataforma premium diseñada para capturar, preservar y estructurar conocimiento humano. El diseño debe transmitir confianza, permanencia, sofisticación y humanidad, evitando estéticas futuristas exageradas o apariencias de chatbot genérico.

Inspiración: Apple, Notion, Linear, Arc Browser, Stripe.

## Tipografía
* **Fuentes Principales**: Inter o SF Pro para legibilidad moderna; Lora o Playfair Display para encabezados editoriales (opcional para dar un toque clásico).

## Paleta de Colores
* **Primario (Azul Profundo)**: `#0A192F` o `#1E293B` - Para fondos oscuros y tipografía principal.
* **Acento (Dorado Sutil)**: `#D4AF37` o `#F59E0B` - Usado con moderación para botones clave, insignias y destaques.
* **Fondo (Blanco Cálido)**: `#FAFAFA` o `#FDFBF7` - Fondo principal que simula papel de alta calidad.
* **Grises Suaves**: `#F3F4F6`, `#E5E7EB` - Para bordes, fondos de tarjetas y texto secundario.

## Componentes y Estilo
* **Botones**: Esquinas suavemente redondeadas, sin sombras excesivas.
* **Tarjetas**: Bordes sutiles de 1px gris suave, sombras muy difuminadas.
* **Formularios**: Minimalistas con etiquetas claras por encima de los campos.
* **Navegación**: Barras laterales limpias, estilo Notion, con mucho espacio en blanco.

## Layout
Diseño espacioso, "editorial", usando márgenes amplios y padding generoso para que el contenido respire. Elevada atención al alineamiento y la cuadrícula.
